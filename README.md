# 1Beauty.Asia - Beauty & Dental Directory (Premium SaaS)
**Hệ thống Danh bạ & Landing Page cao cấp cho ngành Làm đẹp và Nha khoa.**

## 1. Giới thiệu
Dự án **1Beauty.Asia** là nền tảng Directory kết hợp SaaS, cho phép các doanh nghiệp (Spa, Nha khoa, Clinic) đăng ký và sở hữu ngay một Landing Page chuyên nghiệp theo phong cách **Royal Luxury**.

## 2. Các chức năng chính
- **Directory**: Tìm kiếm doanh nghiệp theo khu vực và chuyên môn.
- **SaaS Landing Page**: Tự động tạo trang đích với 3 mẫu template sang trọng.
- **Visual Editor**: Chủ doanh nghiệp tự sửa nội dung trực tiếp trên giao diện.
- **Booking System**: Khách hàng đặt lịch, hệ thống gửi email tự động cho DN và Admin.
- **Subscription**: Dùng thử 30 ngày, tự động ẩn trang khi hết hạn.

## 3. Cấu trúc thư mục
- `/src/app`: Chứa các route chính (Directory, Dashboard, Auth, Landing Pages).
- `/src/components/templates`: Chứa các mẫu giao diện RoyalClassic, ModernMedical...
- `/supabase`: Chứa các bản migration và schema database của dự án.
- `/docs`: Chứa PRD, System Specs và Style Guide chính thức.

## 4. Hướng dẫn chạy nhanh
1. Cấu hình các key trong `.env.local`.
2. Chạy `npm install`.
3. Chạy `npm run dev`.
